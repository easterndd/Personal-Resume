## 二、产品设计——本地工具功能架构

> **前期策略：完全本地化运行。** 无需服务器/域名/备案/支付接口，零固定成本，零被攻击风险，零流量费用。买家在闲鱼下单→把文件发给老大→老大本机跑工具处理→把成品发回去，闲鱼负责收钱和订单管理。

### 2.1 整体技术栈（本地版）

```
界面：Python + Gradio（一行代码出界面，本地浏览器打开，好看又好用）
AI集成：DeepSeek API / 小米MiMo API（只需配置 API Key，本地调用）
文件处理：python-docx（Word读写）、pdfplumber（PDF读取）、reportlab 或 weasyprint（PDF生成）
运行方式：python app.py → 浏览器打开 http://localhost:7860
存储：本机文件夹，处理完手动删除即可
成本：仅 AI API 调用费（每单约 ¥0.1-0.5）
```

**为什么选 Gradio？**
- 3分钟出一个完整的带UI的工具，不用写前端
- 支持文件上传/下载，天然适合这个场景
- 本地跑，不上网，买家数据安全

### 2.2 核心功能模块

#### 模块一：简历上传与解析
- 支持上传格式：`.doc` `.docx` `.pdf` `.txt`
- 自动提取文本内容（python-docx 解析 Word，pdfplumber 解析 PDF）
- 解析后在页面展示原始简历内容供用户确认
- 失败兜底：上传解析失败时，支持用户直接粘贴文字

#### 模块二：简历信息表单（可选填写）
用户可以补充以下信息帮助AI更精准优化：
- 目标岗位 / 目标公司类型（大厂 / 外企 / 国企 / 创业公司）
- 工作年限 / 应届 or 社招
- 核心优势（用户自述）
- 希望突出的项目/技能
- 期望薪资范围（可选）

#### 模块三：AI智能优化引擎（核心）
集成多个大模型，支持一键切换：

| 模型 | 特点 | 适用场景 |
|------|------|---------|
| DeepSeek-V3 | 中文理解强，免费额度多 | 主力模型，日常使用 |
| 小米MiMo | 推理能力强，格式规范 | 逻辑性强的技术岗简历 |
| GPT-4o | 英文简历/外企投递 | 英文岗位、外企简历 |

**AI优化提示词模板（Prompt工程）：**
```
你是一位专业的HR顾问和简历优化专家，有10年招聘经验。
请对以下简历进行优化，要求：
1. 工作经历使用STAR法则（情境-任务-行动-结果），量化成果（加入数字/百分比）
2. 技能描述精准匹配目标岗位：{目标岗位}
3. 语言简洁有力，去除口水话
4. 突出核心竞争力，弱化不相关经历
5. 格式规范，建议保持一页（应届生）或两页（3年+）
6. 输出完整的优化后简历文本，并在最后附上"优化说明"（告知改了什么、为什么改）

原始简历内容：
{简历内容}

目标岗位：{岗位}
```

**可提供的优化服务档次：**
- 基础版：语言润色、错别字修正、格式规范
- 标准版：STAR法则重写工作经历、关键词优化（ATS友好）
- 高级版：行业定制化建议 + 多稿修改 + 求职信赠送

#### 模块四：简历模板库
提供10-20套精美模板，分类：
- 互联网/科技类
- 金融/咨询类
- 应届生通用
- 创意/设计类
- 管理/高管类

模板格式支持 `.docx` 导出，用户可以直接编辑。

#### 模块五：输出与导出
- 在线预览优化后的简历
- 一键下载：`.docx` / `.pdf` 两种格式
- PDF生成方案：
  - 方案A：python-docx 生成 Word，再用 LibreOffice headless 转 PDF
  - 方案B：HTML渲染 → 用 Puppeteer 生成 PDF（排版更精美）

#### 模块六：订单管理后台（老大专用）
- 查看所有提交记录
- 标注订单状态（待处理/已优化/已交付）
- 手动干预和二次润色
- 导出交付文件

## 四、运营流程 SOP（本地版）

```
1. 买家在闲鱼下单 / 私信咨询
        ↓
2. 发送「信息收集表」（目标岗位、工作年限、希望突出的点）
        ↓
3. 买家把简历文件发来（闲鱼聊天传文件 / QQ / 微信）
        ↓
4. 老大把文件拖入本地工具，填写岗位信息，点击"AI优化"
   （工具自动解析→调用DeepSeek→生成优化版）约1-3分钟
        ↓
5. 老大检查AI输出，手动润色调整（5-15分钟）
        ↓
6. 一键导出 docx + pdf，通过闲鱼聊天发回买家
        ↓
7. 请求好评，引导复购/转介绍
        ↓
8. 本地删除买家文件，保护隐私
```

**订单管理完全走闲鱼：**
- 收款：闲鱼平台，无需自己接支付
- 纠纷：闲鱼客服介入，有保障
- 评价：积累好评提升排名
- 无需开发任何订单系统、无需申请微信/支付宝商户
